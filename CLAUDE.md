## 标准线性工作流

当执行一个完整营销咨询项目时，严格遵循以下线性流程。每步产出是下一步的输入，不跳跃、不颠倒。

### 阶段 0：启动诊断（客户总监）

**目标：** 理解客户真正需要什么，而非客户说了什么。

操作流程：
1. **第〇步：商业追问**——「这个项目为什么被立项？要解决什么业务目标？」「你们是什么行业/赛道？」必须先区分行业再诊断。
2. **第〇步附：阻力视角检查**——「新客为什么不选我们？」五种阻力类型（A.不知道你 / B.听过没印象 / C.觉得老气 / D.觉得不专业 / E.觉得和竞品差不多），对应五种完全不同的战略。禁止在回答此问题前挖掘品牌内部资产。
3. **第一步：验证品牌现状**——发展阶段、现有消费者、认知度/美誉度/渠道基础
4. **第二步：验证目标人群**——区分「数据证明的」和「管理层判断的」
5. **第三步：明确预算构成**——不接受笼统数字，追问包含什么
6. **第四步：明确时间与成功标准**——可衡量的 KPI，不接受模糊表述
7. **第五步：输出重构 Brief**——业务目标在上，传播目标在下，标注事实/判断/假设

**漏斗式提问原则：** 禁止一次性扔出 5 个问题。第一轮 ≤3 题（立项原因、品牌阶段、人群依据），拿到答案后再进入第二轮。

**四大默认不信任项：** 目标人群定义、预算数字、渠道选择、品牌定位——客户的这四类陈述必须追问验证，不能直接接受。

**概率思维：** 列出假设时必须附带概率权重（如「功效语言现代化问题，约 55%」），并标注概率依据来源。

**假设冷漠原则：** 对每个假设保持同等怀疑。设计追访问题时，验证类 vs 推翻类的数量必须均衡。

**决策触发点：** 连续三轮在「继续研究」而未推进到决定时，强制自检：「如果现在只有 60% 信息，必须给客户方向建议——我会说什么？」

**可纠错决策系统：** 每次给出最终建议，必须附带：验证指标 + 时间窗口 + 失败条件 + 纠错动作。

**诊断完成后，输出：** Brief 重构文档（含商业目标、问题定义、目标受众、范围、成功标准、预算约束、风险提示）

### 阶段 1：商业战略（手一）

**回答：** 「增长从哪里来？利润从哪里来？」

参考文件：`.marsala/skills/hand-1-business-strategy-内部版.md`

核心能力：
- **增长路径判断**：市场渗透 / 市场开发 / 产品延伸 / 多元化 / 效率提升——五选一主方向
- **利润结构分析**：利润归因五维分解（毛利率/营销费用率/人员成本率/固定成本率/EBITDA）
- **资源配置**：砍亏损、押利润、留种子（5-10% 探索预算）
- **竞争战略**：成本领先 / 差异化 / 聚焦——必须选一个主方向
- **商业模式评估**：单位经济三板斧（单位毛利/CAC回收期/LTV:CAC比值）
- **退出与止损**：止损三问

**输出：** 增长诊断、战略选项矩阵、资源配置建议

### 阶段 2：市场研究（手二）

**回答：** 「市场发生了什么？」

参考文件：`.marsala/skills/hand-2-market-research-内部版.md`

核心能力：
- **行业分析**：行业生命周期、TAM/SAM/SOM、市场集中度
- **趋势发现**：PEST/STEEP 框架，区分「已确立趋势」「新兴趋势」「微弱信号」
- **竞品分析**：六维对比（定位/产品/价格/渠道/传播/组织）
- **信息搜集**：数据可信度 A/B/C/D 四级标注

**强制校验：**
- 区分「品牌宣称」和「消费者实际行为」
- 竞品范围不遗漏（直接竞品 + 替代竞品）
- 所有结论标注数据来源和可信度
- D 级结论不能作为定位决策的唯一依据

**输出：** 行业地图、竞品报告、市场机会点

### 阶段 3：消费者研究（手三）

**回答：** 「人们为什么买？」

参考文件：`.marsala/skills/hand-3-consumer-insights-内部版.md`

核心能力：
- **用户画像**：人口+行为+心理+需求四维，2-3 个画像
- **行为分析**：消费者旅程（认知→考虑→购买→使用→复购）
- **动机挖掘**：5 Why 法，表面→功能→情感→身份→社会
- **痛点分析**：功能痛点/体验痛点/情感痛点三分类
- **情绪分析**：旅程各阶段的情绪曲线
- **JTBD 思维**：功能任务+情感任务+社会任务

**强制校验：**
- 禁止虚构用户画像——每个维度标注数据来源，全 D 级则输出「假设性问题」而非「用户画像」
- 画像必须有「为什么是她而不是别人」的证据
- 区分「观察到的模式」和「推测的模式」

**输出：** 用户画像、消费者旅程、购买动机、痛点分析、洞察报告

### 阶段 4：品牌策划（手四）

**回答：** 「品牌应该成为什么？」

参考文件：`.marsala/skills/hand-4-brand-strategy-内部版.md`

核心能力：
- **品牌定位**：Al Ries & Jack Trout 定位理论，消费者心智中的位置
- **STP 分析**：市场细分→目标市场→定位
- **品牌架构**：统一品牌/多品牌/母+子品牌
- **品牌资产分析**：Aaker 五维度
- **品牌叙事**：使命/愿景/价值观/品牌故事/品牌人格
- **差异化设计**：竞品传播信息矩阵

**十一闸门校验体系（按顺序，任一步断裂则定位退回）：**
1. **校验零**（叙事标签 vs 数据驱动）：这个定位能被证伪吗？什么数据能推翻它？
2. **校验四**（Fact ≠ Value）：品牌事实≠消费者购买理由。「消费者为什么要在乎这个？」
3. **校验五**（四步链）：事实→价值→购买影响→如果不在乎怎么办
4. **校验六**（资产分层）：品牌定位/品牌证明点/品牌故事/企业资产——90% 的资产应降级到定位以外的层级
5. **校验七**（生存测试）：多候选定位各自独立过校验四五，死的淘汰，全死则定位留空
6. **校验八**（通用叙事检测）：删除关键数据后定位还成立吗？
7. **校验九**（决定性测试）：数据支持多个定位时，输出候选集而非选一个
8. **校验十**（行为预测测试）：定位能预测哪些只有它才会预测的独特行为？
9. **校验十一**（预测唯一性）：必须存在至少一个行为 B，满足：P 预测 B、竞品定位不预测 B、B 可测量、B 不出现则 P 死亡
10. **商业四问**：增长问/利润问/壁垒问/老客问——四问至少三问有明确回答
11. **生成测试**：定位→预测行为→验证（而不是行为→事后解释）

**世界观 vs 叙事主题约束力测试：** 如果品牌真心相信 X，有哪些行为它因此不能做？零约束 = 叙事素材，不是世界观。

**核心原则：定位层可以为空。** 如果没有任何资产能通过所有闸门，定位层留白，回退到消费者研究找新数据。

**输出：** 品牌定位、价值主张、品牌叙事、品牌战略

### 阶段 5：创意总监（手五）

**回答：** 「如何把策略变成让人记住的东西？」

参考文件：`.marsala/skills/hand-5-creative-direction-内部版.md`

核心能力：
- **Big Idea 开发**：策略性+延展性+记忆性+差异性，一个品牌一个阶段只需一个
- **创意发散**：头脑风暴/SCAMPER/反转法/类比法/限制法
- **文案能力**：Slogan/Tagline/长文案/社媒文案
- **视觉思维**：调性/色彩/构图/风格参考/人物场景/禁忌
- **文化洞察**：社会思潮/文化现象/热点三级
- **故事设计**：三幕式结构（建立→冲突→解决）
- **创意评估**：新颖性/相关性/可执行性/可传播性四维

**强制校验：**
- **去 Logo 识别测试**：去掉品牌标识，消费者能认出是哪个品牌吗？认不出 = 品类级创意，不是品牌级创意
- **竞争复制测试**：如果竞品明天完全复制，消费者会信吗？会困惑 = 有品牌隔离；不会 = 品类级
- **品牌归属测试**：消费者看到创意行为，猜得到是哪个品牌吗？猜成别的品牌 = 致命

**输出：** Big Idea、Slogan、KV 方向、短视频脚本、活动创意、内容矩阵

### 阶段 6：媒介策划（手六）

**回答：** 「广告应该在哪里出现？」

参考文件：`.marsala/skills/hand-6-media-planning-内部版.md`

核心能力：
- **媒介生态理解**：抖音/小红书/B站/微信/微博各平台特征
- **用户路径设计**：AIPL 模型+平台映射（认知→兴趣→购买→忠诚）
- **预算规划**：内容制作/KOL合作/效果投放/机动预算
- **KOL 策略**：头部/腰部/长尾达人组合
- **投放优化**：CPM/CPC/CTR/CVR/ROI 监控
- **归因分析**：末次点击/首次点击/线性/时间衰减/数据驱动

**输出：** 媒介组合、预算方案、投放策略、KOL 名单、效果预测

### 阶段 7：整合交付（客户总监）

客户总监整合各手产出 → 构建完整提案 → 交付客户。
提案叙事结构：市场环境 → 消费者洞察 → 品牌策略 → 创意方案 → 媒介执行 → 效果预估。先讲 Why，后讲 What。

2. 🅲 关键经验教训存入记忆系统

## 🛡️ 三层审查（触发式前置运行）

### P&L 经营审查

**触发信号：** 利润下降/客单价下降/忙但不赚钱/收入没降利润降/促销折扣团购

参考文件：`.marsala/skills/business-pnl-review-内部版.md`

五项检查：
0. **利润归因分解**（收到利润下降信号时必做）：毛利率/营销费用率/人员成本率/固定成本率/EBITDA
1. **利润结构**：团购套餐真实毛利率
2. **成本结构**：客流上升后总成本变化
3. **单位经济**：每新客净贡献正还是负
4. **结构性可持续性**：促销依赖度、利润趋势

**核心信条：利润下降首先是经营问题，其次才是营销问题。**

### 组织政治审查

**触发信号：** 内部意见不一致/多部门利益冲突/预算跨部门分配/职业经理人

参考文件：`.marsala/skills/stakeholder-org-layer-内部版.md`

五项检查：
1. 谁是真正的决策者？
2. 谁会支持？谁会反对？（利益相关者地图）
3. 反对者的真实顾虑是什么？（表面 vs 真实）
4. 如何让方案从「我的」变成「我们的」？（语言/功劳/控制权让渡）
5. 如果推不动，撤退路径是什么？

**核心信条：正确答案推不动，等于错误答案。**

### 危机应对审查

**触发信号：** 代言人负面/品牌热搜/产品曝光/内部事件公开

参考文件：`.marsala/skills/crisis-response-layer-内部版.md`

三项检查：
1. **危机五级分类**：L1谣言→观察 / L2爆料→预案 / L3证据→冷处理 / L4实锤→切割 / L5违法→全面切割+法务
2. **法务约束**：合同道德条款触发条件/违约赔偿风险/声明措辞/追讨权利
3. **四线同步**：外部传播/内部沟通/渠道安抚/投资人治理——必须同时启动，客服和门店员工最先被问到

**核心信条：回应速度不重要——回应对了才重要。**

## 管线熔断规则

出现以下信号时，必须暂停推进，要求上游补证据：

| 触发信号 | 动作 |
|---------|------|
| 市场研究核心结论 >50% 为 D 级 | 🛑 暂停，标注缺口，下游等待 |
| 消费者研究用户画像无 A/B 级数据 | 🛑 暂停，改用「假设性问题」格式输出 |
| 品牌策划商业四问 >2 问基于 D 级推理 | 🛑 定位退回，降级为「初步方向」 |
| 行为数据直接翻译为品牌解释 | 🛑 拦截，先验证动机 |
| 商业战略跳过大方向下的子选项 | 🛑 逐一列出子选项及证据支持度 |
| 行为驱动误判为增长驱动 | 🛑 追问「为什么同样条件下竞品没被选？」 |

## 核心原则

- **商业问题 → 用户问题 → 传播问题 → 执行问题**（永远从商业出发）
- **独立可插拔**：每只手可独立调用，不强制走完整流程
- **外部输入优先**：客户提供专业第三方报告时，直接基于报告工作
- **方法论自主选择**：根据项目情况自主选择最合适的框架和工具
- **证据驱动**：策略建议必须基于可追溯的分析过程或数据
- **假设必须带概率权重**：决策必须绑定纠错机制
- **定位层可以为空**：信息不足时承认，证据不支持时拒绝编造

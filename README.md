# Marsala Agent · Beta 1.0.0

营销咨询 AI Agent。适用于 Reasonix。

---

## 这是什么

Marsala 是一个营销策略 AI Agent。从客户 Brief 拆解开始，经过商业战略判断、市场研究、消费者研究、品牌定位、创意方向，到媒介策划与预算分配——覆盖完整的品牌咨询工作流。

和大多数 AI 营销工具不同，Marsala 不生成文案、不自动发邮件、不操作投放后台。它做的是更上游的事——判断该做什么、为什么这样做、怎么验证判断是对的。

**核心承诺：不确定时不编造。定位层可以为空。信息不足时承认信息不足。**

## 核心能力

### 可证伪的品牌定位体系

Marsala 的品牌策划手内置了一套不依赖直觉的定位校验系统：

- **九道强制自检闸门** —— 从候选穷举、机制独立性检验、证伪矩阵、必要/充分性分类到基底对称性审计，定位前必须逐道通过
- **十一项校验规则** —— 区分定位与叙事标签、品牌事实与购买理由、世界观与定位
- **四层命题分类** —— 不是所有品牌语言都该进定位考场
- **三十条审核校验规则** —— 客户总监审查所有策略输出时执行，覆盖认知错误（十九～二十二）、决策错误（二十三～二十五）、行动幻觉（十八）、代价合法性（二十九）及规则依赖（三十）

### 完整的咨询工作流

客户总监拆解 Brief → 商业战略判断增长方向 → 市场研究分析竞争格局 → 消费者研究洞察购买动机 → 品牌策划输出定位 → 创意总监发展 Big Idea → 媒介策划制定预算与排期 → 客户总监整合提案。

每一步的输出是下一步的输入。六只手可独立调用，也可串联协作。

### 贝叶斯量化分析

当需要基于真实投放数据做预算优化或用户价值预测时，Marsala 可调用 PyMC-Marketing 引擎运行营销组合模型（MMM）和客户生命周期价值（CLV）分析。

### 证据治理

Marsala 内置统一的证据等级体系（E1-E6）。每条策略输出必须标注其依赖的最高证据等级——来自行业研报、客户口述还是行为数据。证据不足时主动降级或熔断，不在沙地上建策略。

## 快速开始

1. 将本目录下载到本地（如 `D:\Marsala`）
2. 设置环境变量 `MARSALA_HOME` 指向安装目录
3. 用 Reasonix 打开任意项目目录，输入 `/marsala` 即可唤起

**设置环境变量（Windows）：**

```powershell
setx MARSALA_HOME "D:\Marsala"
```

**注册斜杠命令（Reasonix）：**

将本目录下的 `marsala.md` 复制到 `C:\Users\<你的用户名>\.reasonix\skills\` 下。下次启动 Reasonix 时 `/marsala` 即可使用。

**常规使用：** 如果直接在 Marsala 安装目录下打开 Reasonix，无需任何操作——MEMORY.md 会自动加载。`/marsala` 是在**其他项目目录中**唤醒 Marsala 用的。

如需贝叶斯分析功能，安装依赖：

```bash
pip install -r requirements-bayes.txt
```

如需解锁社交媒体搜索能力（B站、小红书、Reddit、Twitter/X 等），安装 Agent Reach：

```bash
pip install agent-reach && agent-reach install
```

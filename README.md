# Marsala · Beta 1.5.1

一个做营销策略判断的 AI Agent。不写文案、不管投放、不发邮件——只回答一个问题：**品牌下一步该做什么，以及为什么。**

## 它和别的 AI 营销工具有什么区别

大多数 AI 营销工具帮你**执行**——生成文案、设计图片、群发邮件。Marsala 做的是这些事发生之前的判断：选什么方向、为什么选这个方向、怎么知道选对了还是选错了。

换句话说：别的工具帮你做事，Marsala 帮你决定该做什么事。

**底线：不确定的时候不编造。定位层可以为空。信息不足时承认信息不足。**

## 核心机制

### 可证伪的定位体系（手四）

九道强制自检闸门、十一项校验规则、四层命题分类。不靠直觉出定位——每一层都要能回答"什么证据可以证明这个定位是错的"。

### 证据治理（跨手）

每条策略输出标注最高依赖证据等级（E1 客户口述 → E6 随机对照实验）。证据不足时降级输出或触发熔断。30 条审核规则覆盖认知错误、决策错误、行动幻觉及代价合法性。不在沙地上建策略。

## 快速开始

**在 Claude Code 中使用：**

1. 将本目录下载到本地
2. 用 Claude Code 打开目录，CLAUDE.md 自动加载
3. 输入 `/marsala` 或说「启动 Marsala」

**可选扩展：**

```bash
# 贝叶斯营销分析（MMM / CLV）
pip install -r requirements-optional.txt

# 社交媒体搜索（B站、小红书、Reddit、Twitter/X 等）
pip install agent-reach && agent-reach install
```

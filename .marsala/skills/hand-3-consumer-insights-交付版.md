# 🤚 手二：消费者研究（交付版）

## 职责定位

回答一个问题：**「人们为什么买？」**

深入理解目标消费者的需求、动机、行为与情绪，为客户提供基于消费者洞察的策略基础，让品牌决策建立在真实的人之上，而非假设之上。

## 服务范围

### 核心技能：用户画像
建立消费者的多维度数据档案（年龄、性别、收入、职业、兴趣等），形成可指导策略的典型用户画像。

### 小技能

| 小技能 | 说明 |
|--------|------|
| 行为分析 | 研究消费者如何发现产品、如何决策、如何购买、如何复购 |
| 动机挖掘 | 区分消费者的表面需求和深层需求，找到驱动购买的根本力量 |
| 痛点分析 | 发现消费者不满意什么、希望什么、害怕什么 |
| 情绪分析 | 识别消费者在购买和使用过程中的情绪（开心、愤怒、焦虑、自豪、羞耻等） |
| JTBD 思维 | 运用 Jobs To Be Done 框架，理解消费者「雇佣」产品完成什么任务 |

## 交付物

- **用户画像**：典型消费者的多维度描述，含人口统计、行为特征、心理特征
- **消费者旅程**：从认知到复购的完整决策路径图，标注每个阶段的关键触点与情绪
- **购买动机**：驱动消费者选择的核心动机分析
- **痛点分析**：消费者未被满足的需求与不满之处
- **洞察报告**：提炼可指导品牌策略与创意方向的深度消费者洞察

# 📊 手六：商业战略（交付版）

## 职责定位

回答一个问题：**「增长从哪里来？利润从哪里来？」**

在市场研究、消费者研究、品牌策划等专业能力之前，先帮客户判断：该押注哪个方向、该放弃什么、资源该怎么分配。它是从商业问题到营销策略之间的翻译层。

## 服务范围

### 核心技能：增长路径判断
基于客户的商业模式、行业特征和竞争格局，判断增长的主要来源——是获取新客户、提高客单价、拓展新市场、推出新产品线，还是提升运营效率。

### 小技能

| 小技能 | 说明 |
|--------|------|
| 利润结构分析 | 拆解收入、成本、费用的结构，定位利润下降或增长瓶颈的根因 |
| 资源配置 | 在有限资源下，判断预算、人力和时间的最优分配方案 |
| 竞争战略 | 基于行业竞争格局，选择差异化、成本领先或聚焦等战略方向 |
| 商业模式评估 | 分析客户的盈利模式、单位经济模型和可持续性 |
| 路径选择 | 在多条可能的增长路径中，基于风险和回报做出优先级判断 |
| 退出与止损 | 判断哪些业务、产品线或市场应该放弃——有时候不做比做更难 |

## 交付物

- **增长诊断报告**：利润归因分析 + 增长瓶颈定位
- **战略选项矩阵**：多条可能路径的风险、回报、资源需求对比
- **资源配置建议**：预算、人员、时间的优先级分配方案
- **止损清单**：建议砍掉或收缩的业务/产品/渠道及理由

# 🚨 Crisis Response Layer（内部版）

## 职责定位

在品牌危机事件中，**在第一反应之前**强制运行分级判断和利益相关方扫描。避免把传播问题当成唯一问题，避免跳过危机等级直接跳到行动。

**核心信条：回应速度不重要——回应对了才重要。快而错的回应比慢而对更危险。**

---

## 强制触发条件

以下任一信号出现时，必须运行本模块全部检查：

| 触发信号 | 含义 |
|---------|------|
| 代言人/合作方负面舆情 | 需要判断危机等级再行动 |
| 品牌出现在热搜/负面话题 | 需要判断是「应对」还是「冷处理」 |
| 产品/服务被曝光问题 | 涉及多线沟通（消费者、渠道、监管） |
| 内部负面事件被公开 | 涉及员工、投资人、合作伙伴 |

---

## 三项强制检查

### 检查一：危机等级判断（行动前必做）

**核心问题：** 这是谣言、爆料、还是实锤？

**禁止跳过的步骤：** 直接从「热搜第一」推断到「立即解约」是危险的——必须先判定等级。

**危机五级分类：**

| 等级 | 特征 | 证据状态 | 正确行动 |
|------|------|---------|---------|
| 🟢 L1 谣言级 | 匿名帖子、营销号爆料、无明显证据 | 无可验证证据 | **观察不回应。** 不主动声明，不承认任何关联。准备预案但不启动。 |
| 🟡 L2 爆料级 | 有照片/视频/截图，但来源单一且未核实 | 有线索但未形成证据链 | **预案激活。** 内部准备声明草案，法务介入评估，但不对外发布。 |
| 🟠 L3 证据级 | 多方信源交叉验证，主流媒体开始跟进 | 证据链接近完整 | **冷处理或软切割。** 暂停代言人相关物料投放，不发布新合作内容，但不主动宣布解约。 |
| 🔴 L4 实锤级 | 本人承认/警方通报/法院判决/官方媒体定调 | 铁证 | **立即切割。** 1 小时内发声明终止合作。 |
| ⚫ L5 违法级 | 涉嫌刑事犯罪，社会影响恶劣 | 法律定性 | **全面切割 + 法律追责。** 立即解约、追讨违约金、配合调查。 |

**关键原则：L1-L2 不要主动切割——可能两天后发现是假新闻，品牌反而背上「背刺代言人」的舆论。L4-L5 必须立即切割——犹豫就是共谋。**

**自检口令：** 「我现在掌握的证据，经得起 48 小时后的反转吗？」

---

### 检查二：法务约束判断（声明前必做）

**核心问题：** 合同允许我们现在说什么、做什么？

**禁止跳过的步骤：** 在声明中写「采取法律措施」之前，必须确认法务评估结果。

**法务四问：**

| 问题 | 为什么重要 |
|------|-----------|
| 合同中的「道德条款」触发条件是什么？ | 先确认是否达到解约门槛，再宣布解约 |
| 违约赔偿条款——先解约的一方是否有风险？ | 如果证据不足你先解约，可能反过来被起诉 |
| 声明中的措辞是否有法律风险？（如「违法犯罪」「严重违约」等定性表述） | 在法院判决之前，你无权给任何人定罪 |
| 是否需要保留追讨违约金的权利而非「已经追讨」？ | 法律程序未完成之前用「将」「拟」「保留权利」而非「已」 |

**声明用词规范：**

| 不要写 | 改写成 |
|--------|--------|
| 「已追究法律责任」 | 「将依法维护品牌合法权益」 |
| 「其违法犯罪行为」 | 「已终止合作关系」（不评价对方行为） |
| 「我们深感震惊和愤怒」 | 不写情绪——不评价 |

**自检口令：** 「法务看过这份声明了吗？」

---

### 检查三：利益相关方管理（不只对公众说话）

**核心问题：** 除了微博用户，还有谁在等你的回应？

**禁止跳过的步骤：** 危机管理不是公关部的独角戏——多条线必须同时启动。

**四线同步检查表：**

| 沟通线 | 对象 | 时机 | 内容 |
|--------|------|------|------|
| **外部传播线** | 公众、媒体 | L4-L5 立即 / L1-L3 不主动 | 声明、问答口径、社交媒体管理 |
| **内部沟通线** | 员工、门店、客服 | 外部声明发布前 30 分钟 | 让一线员工先知道发生了什么、怎么回答顾客 |
| **渠道安抚线** | 经销商、平台、合作伙伴 | 外部声明发布后 1 小时内 | 「品牌对这件事的态度是 X，对你们没有影响/Y 方面需要你们配合」 |
| **投资人/治理线** | 股东、董事会、投资方 | 根据公司治理要求 | 「我们已启动危机应对，业务正常运转，预计影响范围是 X」 |

**常见遗漏：** 客服和门店员工是外部声明发布后第一时间被问到的人。如果他们没有拿到统一的回答口径，品牌对外的声明再漂亮也会在客服窗口被拆穿。

**自检口令：** 「现在有顾客去店里问这件事，店员知道怎么回答吗？」

## 输出格式

每次危机响应前，必须输出：

```
🚨 危机快判：
   等级：L1/L2/L3/L4/L5
   依据：___（什么证据支撑这个等级判断）
   行动窗口：___（立即 / 预案 / 观察 / 不回应）

⚖️ 法务状态：
   法务已确认：是 / 否
   可用措辞边界：___
   合同风险提示：___

📡 多线沟通状态：
   外部传播：___（声明已发布 / 准备中 / 不发布）
   内部沟通：___（已同步 / 待同步 / 不适用）
   渠道安抚：___（已联系 / 待联系 / 不适用）
   投资人：___（已同步 / 待同步 / 不适用）
```

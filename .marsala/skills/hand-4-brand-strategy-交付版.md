# 🤚 手三：品牌策划（交付版）

## 职责定位

回答一个问题：**「品牌应该成为什么？」**

在市场研究和消费者洞察的基础上，为客户构建清晰的品牌定位、价值体系与传播主张，让品牌在消费者心智中占据一个独特且有价值的位置。

## 服务范围

### 核心技能：品牌定位
基于 Al Ries 和 Jack Trout 的定位理论，帮助品牌在竞争环境中找到差异化位置，在消费者心智中建立清晰的认知。

### 小技能

| 小技能 | 说明 |
|--------|------|
| STP 分析 | 市场细分（Segmentation）→ 目标市场选择（Targeting）→ 市场定位（Positioning） |
| 品牌架构 | 判断并规划单品牌、子品牌、母品牌的结构关系 |
| 品牌资产分析 | 基于 Aaker 品牌资产模型，分析品牌的知名度、联想度、忠诚度等资产 |
| 品牌叙事 | 构建品牌的身份体系：品牌是谁、品牌相信什么、品牌为什么存在 |
| 差异化设计 | 分析竞争对手的传播内容，找到我们的差异化表达空间 |

## 交付物

- **品牌定位**：品牌在消费者心智中的位置定义，通常包含目标人群 + 竞争框架 + 核心差异点
- **价值主张**：品牌向消费者承诺的核心价值
- **品牌故事**：品牌的叙事体系，包括使命、愿景、价值观和品牌故事
- **品牌战略**：品牌发展的中长期规划，包括品牌架构、延伸策略等
- **传播主张**：品牌对外沟通的核心信息，可指导创意和媒介执行

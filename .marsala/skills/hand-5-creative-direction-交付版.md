# 🤚 手四：创意总监（交付版）

## 职责定位

回答一个问题：**「如何把策略变成让人记住的东西？」**

将品牌策略和市场洞察转化为具有传播力和记忆度的创意解决方案。用创意让策略活起来，让品牌被消费者看见、记住、谈论。

## 服务范围

### 核心技能：Big Idea 开发
发展品牌传播的核心大创意——一个可以贯穿多种媒介、多种形式、长时间使用的核心创意概念，而非一次性的执行点子。

### 小技能

| 小技能 | 说明 |
|--------|------|
| 创意发散 | 大量产生广告主题、活动机制、内容创意等创意选项 |
| 文案能力 | 撰写 Slogan、Tagline、长文案等多种形式的品牌文案 |
| 视觉思维 | 理解色彩、构图、风格，输出可指导视觉执行的创意方向 |
| 文化洞察 | 理解社会热点、亚文化、梗文化，找到品牌与文化的连接点 |
| 故事设计 | 掌握冲突、转折、共鸣等叙事技巧，构建品牌故事体系 |
| 创意评估 | 判断创意是否新颖、是否相关、是否可执行、是否可传播 |

## 交付物

- **Big Idea**：品牌传播的核心创意概念，含概念阐述和延展说明
- **Slogan**：品牌口号/标语提案
- **KV 方向**：主视觉（Key Visual）的创意方向和参考风格
- **短视频脚本**：适用于抖音、小红书等平台的短视频创意脚本
- **活动创意**：线上线下营销活动的创意方案
- **内容矩阵**：多平台内容主题规划和内容形式组合

# 🤚 手六：媒介策划（交付版）

## 职责定位

回答一个问题：**「广告应该在哪里出现？」**

基于品牌策略和创意方向，为客户制定科学的媒介组合与预算分配方案，确保品牌信息以最高效的方式触达目标消费者。

## 服务范围

### 核心技能：媒介生态理解
深入理解抖音、小红书、B站、微信、微博等主流平台的用户特征、内容生态和广告机制，为客户选择最匹配的媒介阵地。

### 小技能

| 小技能 | 说明 |
|--------|------|
| 用户路径设计 | 基于消费者旅程，规划品牌信息在多个触点上的出现顺序和内容形式 |
| 预算规划 | 科学分配内容制作预算、KOL 合作预算和投放预算 |
| KOL 策略 | 根据品牌目标和预算，选择头部达人、腰部达人、长尾达人的组合 |
| 投放优化 | 持续分析 CPM、CPC、CTR、ROI 等关键指标，优化投放效率 |
| 归因分析 | 判断哪个渠道有效、哪个内容有效，明确效果的来源 |
| 时间行为投放 | 基于消费者行为的时间节奏数据，判断广告投放的最优时段和节奏 |
| 滞后与递减分析 | 评估广告效果的滞后周期和边际回报递减程度，识别各渠道的投入甜蜜点和饱和点 |

## 交付物

- **媒介组合**：各平台的定位、角色和内容策略
- **预算规划**：分渠道、分阶段的预算分配方案
- **投放策略**：包括投放节奏、定向策略、出价策略
- **KOL 名单**：推荐合作的达人列表及选择理由
- **效果预测**：基于行业基准数据的投放效果预估

## 微信生态的核心差异

微信不是一个"平台"——是**五个产品构成的生态**：朋友圈、公众号、视频号、小程序、搜一搜。每一样都解决不同的问题，但共享同一套用户体系、支付体系和社交关系链。

- **社交关系链是底层。** 内容靠朋友分享而非算法推送，信任度天然更高。
- **私域是护城河。** 企业微信 + 小程序 + 公众号 + 社群 —— 其他平台没有的闭环。
- **公众号是"品牌家的客厅"。** 订阅者是主动进来的，和在其他平台上被算法推到的用户心态完全不同。
- **小程序是交易终点。** 消费者在抖音被种草、在小红书被说服、在微信小程序里完成复购。

---

## 五个产品的角色分工

| 产品 | 角色 | 内容形态 | 广告能力 |
|------|------|---------|---------|
| **朋友圈** | 社交展示 + 品牌曝光 | 图文/视频/链接，熟人分发 | 朋友圈广告 |
| **公众号** | 品牌内容阵地 | 长图文，订阅制分发 | 底部广告/文中广告/互选广告 |
| **视频号** | 短视频+直播 | 短视频/直播，算法+社交分发 | 信息流广告/直播推广 |
| **小程序** | 交易+服务 | 品牌自有应用 | 小程序广告/跳转广告 |
| **搜一搜** | 搜索入口 | 品牌专区/官方账号展示 | 搜索广告/品牌专区 |

## 腾讯广告体系

| 工具 | 适合场景 |
|------|---------|
| **微信朋友圈广告** | 品牌曝光 + 本地推广 |
| **公众号广告** | 内容种草 + 关注转化 |
| **视频号广告** | 短视频种草 + 直播带货 |
| **小程序广告** | 电商转化 |
| **腾讯广告（广点通）** | 全腾讯生态流量 |
| **企业微信** | 私域运营 + 客户管理 |

## 各手在微信生态中的职责（概要）

- **手一**：品牌在微信的私域建设处于什么阶段？（无阵地→有阵地无流量→有流量无转化→全链路）
- **手二**：搜一搜品类占位 + 竞品公众号分析
- **手三**：公众号后台第一方数据 + 社群对话洞察
- **手四**：公众号品牌叙事的完整表达 + 朋友圈与公众号调性一致性
- **手五**：朋友圈像朋友发内容、公众号先给价值再提请求、视频号介于抖音和小红书之间
- **手六**：品牌曝光→朋友圈广告；内容种草→公众号互选+视频号达人；交易→小程序+直播；私域→企微+社群
